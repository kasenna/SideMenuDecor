# SideMenuDecor
An Android library for decorating a DrawerLayout.

Supported API: 14+

##Set up<br>

Just add this to your build.gradle:
```java
repositories {
    mavenCentral()
}

dependencies {
    compile 'com.github.kasenna:SideMenuDecorLibrary:1.0.0'
}
```
